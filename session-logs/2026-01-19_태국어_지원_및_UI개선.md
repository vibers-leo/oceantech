# 태국어 지원 및 모바일 UI 개선

## 작업 개요

- **일시**: 2026-01-19
- **주요 작업**: 태국어(TH) 버전 추가, 상세페이지 이미지 로직 구현, 모바일 메뉴 UI 개선, 빌드 오류 수정

## 상세 작업 내역

### 1. 태국어(TH) 지원 추가

- **이미지 리소스 정리**: `public/th` 폴더 내의 이미지 파일들을 `rminu-th_01.jpg` ~ `rminu-th_13.jpg` 순서로 정렬 및 이름 변경.
- **번역 데이터 추가**: `src/lib/translations.ts`에 태국어(`th`) 키 및 번역 데이터 추가 (기본적으로 영어 기반으로 설정).
- **타입 정의 업데이트**: `Language` 타입에 `'th'` 추가.

### 2. 상세페이지 로직 개선 (`src/components/ProductDetail.tsx`)

- 태국어 선택 시 `public/th` 폴더의 이미지를 로드하도록 로직 수정.
- 태국어 버전의 이미지 개수(13장)와 확장자(.jpg) 처리 로직 구현.

### 3. 헤더 및 모바일 UI 개선 (`src/components/Header.tsx`, `Header.module.css`)

- **데스크탑**: 헤더 우측 상단 언어 선택기에 `TH` 버튼 추가.
- **모바일**:
  - 좁은 화면(`900px` 이하)에서는 상단 언어 버튼을 숨김 처리.
  - 햄버거 메뉴 오픈 시 **최상단**에 언어 선택 버튼(KR | EN | TH)을 배치하여 접근성 강화.
  - 선택된 언어에 강조 스타일(Bold, Underline) 적용.

### 4. 빌드 오류 수정 (`src/app/presentation/overseas-2026/page.tsx`)

- 프레젠테이션 페이지의 `content` 객체에 `th` 키가 누락되어 발생한 빌드 타입 에러 해결.
- 영어 콘텐츠를 복제하여 태국어 섹션 임시 추가 (추후 번역 데이터로 교체 가능).

## 결과

- 홈페이지 및 상세페이지에서 태국어 버전이 정상적으로 표시됨.
- 모바일 환경에서 사용자가 더 편하게 언어를 변경할 수 있도록 UI가 개선됨.
- 전체 프로젝트 빌드 및 배포 정상화.
